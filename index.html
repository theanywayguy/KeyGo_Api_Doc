Introduction

Welcome to the KeyGO API documentation. This API provides endpoints for managing accounts, cars, rentals, favorites, and reviews for a car rental service. It follows RESTful principles and uses JWT for authentication.

Authentication

The API uses Bearer token authentication. Include a valid JWT in the Authorization header:

Authorization: Bearer {your-token}

Endpoints

Account

POST /api/account/login

Authenticates a user and returns a JWT token.

Request Body
{ "username": "string", "password": "string" }
Responses

200: Successful login

POST /api/account/register

Registers a new user.

Request Body
{ "username": "string", "emailAddress": "string", "password": "string" }
Responses

200: Successful registration

POST /api/account/assign-role

Assigns a role to a user.

Request Body
{ "username": "string", "role": "string" }
Responses

200: Role assigned successfully

Car

GET /api/car

Retrieves a list of cars with optional filters.

Query Parameters
  • carMake: string (optional)
  • carModel: string (optional)
  • IsAvailable: boolean (optional)
  • Year: integer (optional)
  • Minprice: integer (optional)
  • Maxprice: integer (optional)
  • isDescending: boolean (optional)
  • PageNumber: integer (optional)
  • Pagesize: integer (optional)
Responses

200: List of cars

POST /api/car

Creates a new car entry.

Request Body
{ "make": "string", "model": "string", "year": 0, "pricePerDay": 0, "isAvailable": true }
Responses

200: Car created successfully

GET /api/car/{id}

Retrieves a car by its ID.

Path Parameters
  • id: integer (required)
Responses

200: Car details

PUT /api/car/{id}

Updates a car by its ID.

Path Parameters
  • id: integer (required)
Request Body
{ "make": "string", "model": "string", "year": 0, "pricePerDay": 0, "isAvailable": true }
Responses

200: Car updated successfully

PATCH /api/car/{id}

Partially updates a car by its ID.

Path Parameters
  • id: integer (required)
Request Body
{ "make": "string", "model": "string", "year": 0, "pricePerDay": 0, "isAvailable": true }
Responses

200: Car patched successfully

DELETE /api/car/{id}

Deletes a car by its ID.

Path Parameters
  • id: integer (required)
Responses

200: Car deleted successfully
